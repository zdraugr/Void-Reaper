# Void Reaper

Void Reaper is a Vampire-Survivors-style space survival roguelite packed into a single, self-contained HTML file with zero dependencies — all sprites, sound, and starfield are generated at runtime. You only move; your weapons auto-fire. Survive an endless swarm, collect energy to level up, draft and evolve weapons, and bank gold between runs to unlock new ships.

## How to play
- **Move:** WASD or arrow keys (weapons fire automatically)
- **Dash:** Space (Afterburner — only when that passive is owned)
- **Pick an upgrade:** press 1 / 2 / 3 or click the card
- **Pause:** P · **Mute:** M

Survive as long as you can, build a synergistic loadout, and chase a better time.
